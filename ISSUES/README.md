# Gaggimate Issues & Bugs

This folder is for reporting bugs, issues, and problems found in the Gaggimate project. 

## How to Report an Issue

1. Create a new markdown file in this directory with a descriptive name (e.g., `brew-timer-offset.md`)
2. Use the template below to fill in the details
3. Commit and push your issue report to the branch
4. Team will review and address reported issues

## Issue Template

```markdown
# [Bug Title]

**Date Reported:** YYYY-MM-DD  
**Reported By:** [Your name/username]  
**Status:** Open / In Progress / Fixed

## Description
Brief description of what is not working correctly.

## Expected Behavior
What should happen instead?

## Current Behavior
What is currently happening?

## Steps to Reproduce
1. Step one
2. Step two
3. Step three

## Affected Area
- [ ] Brew Screen
- [ ] Steam Screen
- [ ] Water Screen
- [ ] Grind Screen
- [ ] Profile Screen
- [ ] Status Screen
- [ ] Menu Screen
- [ ] Simulator
- [ ] Hardware

## Severity
- [ ] Critical (blocks functionality)
- [ ] High (significant issue)
- [ ] Medium (noticeable but workaround exists)
- [ ] Low (minor cosmetic issue)

## Screenshots/Evidence
(If applicable)

## Suggested Fix
Describe how this issue might be resolved.

## Additional Notes
Any other relevant information.
```
